Detailed Description

Class designed to calculate and produce a list of MapAreas representing each unique connected area of the map.

The class takes in an IMapOf<bool>, where a value of true for a given position indicates it should be part of a map area, and false indicates it should not be part of any map area. In a classic roguelike dungeon example, this might be a walkability map where floors return a value of true and walls a value of false.

Currently, the class only supports manhattan (4-way) connectivity, but will in the future support 8-way connectivity.
