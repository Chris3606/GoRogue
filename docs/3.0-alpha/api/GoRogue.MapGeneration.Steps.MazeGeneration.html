Remarks

This generation steps generates mazes, and adds the tunnels made to the ItemList<TItem> context component with the proper tag (if one is specified) on the GenerationContext. If no such component exists, one is created. It also sets the all locations inside the tunnels to true in the map's "WallFloor" map view context component. If the GenerationContext has an existing "WallFloor" context component, that component is used. If not, an ArrayMap<T> where T is bool is created and added to the map context, whose width/height match Width/Height.
