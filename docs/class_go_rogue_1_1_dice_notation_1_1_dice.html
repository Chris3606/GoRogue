Detailed Description

The most important DiceNotation class – contains functions to roll dice, and to retrieve an IDiceExpression instance representing a given expression (useful and more efficient if a dice roll is used multiple times).

static

Uses the IParser specified in the DiceParser variable to produce a IDiceExpression instance representing the given dice expression.

Because parsing can add significant time, retrieving an IDiceExpression instance instead of using the Roll function can be useful if a given expression will be rolled multiple times.

Parameters
expressionThe string dice expression to parse.
Returns
An IDiceExpression instance representing the parsed string.

IRandom  random = null 
)
static

Uses the IParser specified in the DiceParser variable to parse the given dice expression, roll it, and return the result.

While more convenient, parsing is computationally more expensive than evaluation. If a dice expression will be rolled many times, it is more efficient to use the Parse method once, and use the resulting IDiceExpression instance to roll the expression each time it is needed.

Parameters
expressionThe string dice expression to parse.
randomRNG to use to perform the roll. If null is specified, the default RNG is used.
Returns
The result of evaluating the dice expression given.
