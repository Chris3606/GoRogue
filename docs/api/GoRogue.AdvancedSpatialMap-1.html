Class AdvancedSpatialMap<T>

Advanced version of SpatialMap that allows for use of a custom IEqualityComparer for hashing and comparison of type T. May be useful for cases where one does not want to implement IHasID, or if you need to use a value type in a SpatialMap. For simple cases, it is recommended to use SpatialMap instead.
