Class AdvancedMultiSpatialMap<T>

Advanced version of MultiSpatialMap that allows for use of a custom IEqualityComparer for hashing and comparison of type T. May be useful for cases where one does not want to implement IHasID, or if you need to use a value type in a MultiSpatialMap. For simple cases, it is recommended to use MultiSpatialMap instead.
