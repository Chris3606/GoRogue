ToString(Int32, Func<T, String>)

Prints the values in the ArrayMap2D, using the function specified to turn elements into strings, and using the "field length" specified. Each element of type T will have spaces added to cause it to take up exactly fieldSize characters, provided fieldSize is less than the length of the element's string represention. A positive-number right-aligns the text within the field, while a negative number left-aligns the text.
