Creates a normal distribution based on a lower and upper value, and a distance to place those points from the mean. The mean is placed precisely in between the upper and lower values given, and the upper and lower values will both be exactly deviationsFromMean away from the mean.

Parameters
lowerLower value by which to define the distribution.
upperUpper value by which to define the distribution.
deviationsFromMeanNumber of deviations from the mean at which to place the lower and upper values given.
Returns
A NormalDistribution constructed such that the mean is precisely in between the lower and upper values given, and the lower and upper values are exatly the specified number of deviations away from the mean.
