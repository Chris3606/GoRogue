Detailed Description

Class representing a method of calculating distance. You cannot create instances of this class using a constructor – instead this class contains static instances representing the applicable types of distance calculations.

Provides functions that calculate the distance between two points according to the distance measurement being used. Distance instances can also be explicitly casted to Radius instances – the 2D radius shape that corresponds to the shape of a "radius" according to the casted distance calculation is retrieved (eg, EUCLIDEAN casts to CIRCLE, MANHATTAN to DIAMOND, etc.)
